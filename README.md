# 🧮Grade Calculator

## 🔍Project Problem
Create a function that takes student scores as input, and outputs grades. Use the following grading criteria for input scores:
- 100 - 70 = A
- 69 - 65 = B
- 64 - 55 = C
- 54 - 45 = D
- 44 - 0 = F

**Requirements:**
1. The function should be able to accept multiple scores as input and print out the corresponding number of grades
2. The use of control flow must be demonstrated
3. Adequate documentation
